CSWD Lab: Bootstrap 4 Simple Page Exercise

  • Your task is to edit the file bs4_simple_page.html, supplied as part of the lab material, to produce the page layout shown in the picture.
  • The rules:
    • you may add an HTML element and nested elements for the navigation bar at the top
    • you may assign HTML elements that have the class attribute to new classes, which means you can:
      • change class="" to class="some-class"
      • change class="some-class" to class="some-class some-other-class"

      Note that an element can be assigned to more than one class!

    • the classes to which you assign the elements must be Bootstrap 4 classes
    • you may not make any other kind of modification to the file
    • you may not use any other files
  • Tips:
    • use the Bootstrap navbar for the nav section (you will find help with this here)
    • note that the navbar-dark class will yield light lettering in the navbar and vice versa
    • make sure that the navbar is expanded for small sized screens and bigger
    • in the main section use appropriate Bootstrap rows and columns to create the layout (information about Bootstrap 4 grid system can be found here)
    • use the Bootstrap 4 margin and padding classes (see section Spacing here) alongside class card to space out the elements
    • use background and text centering classes alongside card-body to style the text in the cards (see classes text-* here)
    • for the footer, you can use the fixed-bottom class, background, text formatting and padding classes to get the desired effect
