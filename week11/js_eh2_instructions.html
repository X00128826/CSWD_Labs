Javascript Event Handlers - Dynamic Setup from Code

Exercise 1: Setting the body colour to an input value

  • A text input element and a button are defined in the frist section in the HTML file lab_eh2.html.
  • Create the required Javascript file (see script element in HTML file for name).
  • Write a function that reads the input value and sets the background of the HTML document's body to the colour specified in the string (e.g. 'red' or 'gray').
  • Set the function as handler for clicks on the button.
  • You must not modify the HTML file at all. This means that you will have to set the event handler dynamically.

Exercise 2: Calculator once again

  • Two number input elements, a button and a paragraph element are defined in the second section in the HTML file lab_eh2.html.
  • Create the required Javascript file if you have not already created it while doing Exercise 1 (see script element in HTML file for name).
  • Implement a calculator that performs the four basic arithmetic operations on the two numbers input by the user and writes the answer into the element belonging to class 'answer'.
  • You must not modify the HTML file at all. This means that you will have to access elements by means other than document.getElementById() and set the four event handlers dynamically.
