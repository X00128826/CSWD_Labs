Javascript Loop Exercises

For these exercises you are given the files loops.html and loops.js. The Javascript file has been included in the HTML file.

Exercise 1: Print Even Numbers

  • In the file loops.js, declare a function called printEven, with one parameter called max.
  • Define the function so that it prints out to the console all even numbers starting at 0 and ending at max (not including max).
  • Use a for loop in the function.
  • Test the function by loading the HTML file and calling the function from the console.

Exercise 2: Spacing out letters

  • In file loops.js, declare a function called appendSpace, with one parameter called str.
  • This function should return a string that consists of the passed-in value in str and a space character concatenated together.
  • In file loops.js, declare a function called spaceOut, with no parameters.
  • Define this function so that it
    • reads the string contained in the input element with id "word"
    • iterates through the characters of the string and for each
      • calls function appendSpace, passing in the character
      • appends the returned value to the paragraph element with id 'spaced'
  • Set function spaceOut as the handler for clicks on the button Space out and test.
  • Video of sample required behaviour can be found here.

Exercise 3: Paragraphs

  • In file loops.js, declare a function called populateTemplate, with no parameters.
  • Define the function so that it
    • reads the number from the input element with id 'num-pgphs' and writes it to the variable numberOfParagraphs
    • uses a for loop to iterate through numbers 1 to numberOfParagraphs
    • at each iteration adds a paragraph element to the element with id 'txt'
    • the paragraph elements should have the text 'Paragraph 1', 'Paragraph 2' etc.
  • Set this function as the handler for clicks on the button Populate story template and test.
  • Video of sample required behaviour can be found here.
