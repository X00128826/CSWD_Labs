Javascript Object Exercises

Exercise 1:

  • Create an HTML file with the following input elements:
    • a text input element with id 'name' and an appropriate label
    • a text input element with id 'surname' and an appropriate label
    • a number input element with id 'year' and an appropriate label
    • a dropdown list with values 'Software Development', 'IT Management', 'Computing with Language', 'Computing with DA' (see here how to create the list)
    • a button with text 'Save data'
    • a button with text 'Create message'
    • a paragraph element with an id
  • Create a Javascript file and include it in the HTML file, then implement the following behaviour in the Javascript file, adding any required handlers in the HTML:
    • When the button 'Save data' is clicked, the data from the input elements should be copied into an object, with each input element's value stored in a separate property of the object. The object should be preserved after the handler completes execution.
    • When the button 'Create message' is clicked a message is created and written into the paragraph element. The message should read: "Hello [1] [2], welcome to year [3] of the [4] programme at TU Dublin in Tallaght!", where [1], [2], [3] and [4] are placeholders for actual values, for example, "Hello Pablo Picasso, welcome to year 1 of the Fine Art programme at TU Dublin in Tallaght!"
