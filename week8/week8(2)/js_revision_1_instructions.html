Javascript Revision Exercises 1

  1. Include Javascript code internally in the HTML file

    Example:
    Add some code internally in the document body of js_ca_practice_workfile.html. The code should post an alert with message 'Hello' and write the same string to the console. Test that your code runs by loading the page.
  2. Include Javascript code externally i.e. from a separate Javascript file

    Example:
    Create a Javascript file called practice.js. Place the file in a folder called scripts in the same folder as js_ca_practice_workfile.html. In the file practice.js, write some code that posts an alert with message 'Hello from practice file!' and writes the same to the console. Include practice.js at the end of the document body of js_ca_practice_workfile.html. Test that your code runs by loading the page.
  3. Use API output functions alert() and console.log()

    Examples:
    1. In practice.js, write a line of code that causes an alert window to pop up when the page is loaded.
    2. In the file js_ca_practice_workfile.html internally include some code that causes the string Hello, hello! to be output to the console while the HTML page is being loaded.
  4. Define a function with no parameters or with one parameter

    Examples:
    1. Define a function called writeInfoToConsole() internally in the file js_ca_practice_workfile.html. The function should write three short lines of information about any topic to the console. Load the html page and run your function in the console to test it.
    2. In practice.js, define a function called doubleTheNumber with one parameter. The function should multiply by 2 the number passed in and write the result to the console. Load the html page and run your function in the console to test it.
  5. Read from an element in the document

    Example:
    Write a function that reads the contents of the paragraph element with id my-text in js_ca_practice_workfile.html and writes it to the console. Load the html page and run your function in the console to test it.
  6. Write to an element in the document

    Example:
    Write a function that sets the string "Ho ho ho here we go" as the contents of the element with id text-destination-1. Load the html page and run your function in the console to test it.
  7. Set up an event handler

    Example:
    Set up an event handler by modifying the HTML file so that when the button Alert! in file js_ca_practice_workfile.html is clicked an alert pops up saying "You've clicked the button!". Test your solution by clicking the button.
  8. Read input from an input element

    Example:
    Write a function that reads the input from the input element with id text-input in js_ca_practice_workfile.html and writes it to the console. Load the html page and run your function in the console to test it.
  9. Create HTML elements in an HTML file

    Examples:
    1. Create a paragraph element in js_ca_practice_workfile.html.
    2. Create an input element of type date in js_ca_practice_workfile.html.
    3. Create a label element with text Date: just in front of the date input element.
  10. Add attributes to HTML elements

    Examples:
    1. Assign the paragraph element created above (in the first example for Create HTML element in an HTML file) to class abc.
    2. Add an id (any value you choose) to the input element created above (in the second example for Create HTML element in an HTML file).
    3. Add the attribute from to the label element created above, so as to associate it with the date input element.
  11. Perform several of the above together

    Examples:
    1. Write a Javascript function internally in the html file js_ca_practice_workfile.html that copies the contents of the element with id my-text to the element with id text-destination-2. Set up an event handler so that when the button Copy my text to destination 2 is clicked that function is called. Test your solution by loading the HTML page and clicking on the button.
    2. Set up an event handler so that when button Double the number is clicked, the number typed into the number input element (to the left of the button) is multiplied by two and the answer displayed in the paragraph element that follows the text "Answer: " (to the right of the button), in the file js_ca_practice_workfile.html. Define a function for this and add any attributes you need onto the button and the paragraph element. Test your solution by loading the HTML page and clicking on the button.
  12. Conditional statements

    Example:
    Working with HTML file js_ca_practice_workfile.html, write a function that toggles the bakground colour of the div element with content My box between lightblue and lightsalmon. The property that controls the background colour of an element is style.background. Set up an event handler that calls this function when the button Toggle colour is clicked. You may add ids or classes to the elements in the HTML file. (The conditional statement in this example is checking what the colour currently is so that the change can be made to the other colour).
